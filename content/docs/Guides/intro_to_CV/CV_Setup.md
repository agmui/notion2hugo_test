# Cloning the repo

[link_preview](https://github.com/Thornbots/CV/)

```bash
git clone https://github.com/Thornbots/CV.git
```

## install python

```bash
sudo apt install python3.8
```

## installing `requirements.txt` packages

```bash
cd CV
python3 -m venv .venv
pip install requirements.txt
source ./.venv/bin/activate
```
